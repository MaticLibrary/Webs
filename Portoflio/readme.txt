Temat: Strona wizytówkowa dla fotografa
Opis:
Chciałbym zamówić prostą, nowoczesną stronę typu one-page, która będzie moją wizytówką jako fotografa ślubnego. Strona powinna zawierać:

    sekcję „o mnie”

    galerię zdjęć z podziałem na kategorie (np. ślub, portret, plener)

    formularz kontaktowy

    integrację z Instagramem (automatyczne ładowanie najnowszych postów)

    responsywny design (mobile-first)

Technologie: HTML, CSS (np. Tailwind), JS, opcjonalnie React
Backend: prosty serwer e-mail do formularza kontaktowego (Node/Express lub inne rozwiązanie)
